# RAML to JAX-RS converter - Gradle Plugin Example

This project demonstrates the usage of the RAML-JAX-RS Gradle plug-in in a Jersey 2 API project.

## Pre-requisites
- JDK6 or better
- Maven 3

## Running and Testing
To generate output execute 
`gradle raml-generate`
